IoT‑Based Flood Prediction System for Agricultural Fields Using ML

Major B.Tech Project – EIE Department

Developed an end-to-end IoT-based Flood Prediction System for agricultural fields using an Arduino-based sensor node integrated with Ultrasonic, Soil Moisture, and DHT11 sensors to continuously monitor environmental conditions. A Raspberry Pi gateway was used for data processing and communication, while LoRa technology enabled long-range and low-power wireless transmission suitable for rural areas. Real-time sensor data such as water level, soil moisture, temperature, and humidity were collected and analyzed using a Random Forest Machine Learning model to predict whether field conditions were normal or flood-prone. The system also included ThingSpeak dashboards and a Flask-based web application for live visualization of sensor readings and prediction results. To improve farmer safety and response time, a GSM module was integrated to automatically send “Flood Alert” SMS notifications during high-risk situations. The project was designed as a low-cost, scalable, and rural-friendly solution capable of operating effectively even in areas with limited internet connectivity.

Technologies: Raspberry Pi, LoRa, GSM, IoT, Machine Learning

Predicting Ethylene Gas Concentration Using ML

Minor B.Tech Project – CSE(AIML & IoT) Department

The project focuses on developing a machine learning–based system to predict ethylene gas concentration using a low-cost gas sensor array. Ethylene is an important plant hormone widely used in agriculture because it directly influences fruit ripening, freshness, and shelf life. Accurate monitoring of ethylene levels helps farmers, storage facilities, and food industries maintain product quality and reduce post-harvest losses. In addition to agricultural applications, the project also addresses industrial safety concerns, as ethylene is highly flammable and can become hazardous at high concentrations. Traditional gas detection systems mainly rely on threshold-based alarms, which only activate after gas levels exceed a fixed limit and are unable to predict future concentration trends. To overcome these limitations, the proposed system uses machine learning techniques to analyze sensor data and estimate ethylene concentration more effectively and intelligently. The project also avoids the need for expensive and specialized gas analyzers by utilizing affordable, general-purpose sensors, making the solution scalable, cost-effective, and suitable for real-world deployment in agricultural storage units and industrial environments.

Technologies: Python, TGS2610 Sensor, Machine Learning
